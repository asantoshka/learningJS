# Reference type

1. Array -> Uses square brackets `[]`
2. Objects -> Uses curly brackets `{}`
3. Functions

```
const myFunction = function(){
    console.log("Hello")
} 
```

> In javascript, we don't declare the variable data types so we can call it as dynamically typed.
